# Contribuindo para o Agent Runner 🚀

Bem-vindo ao **Agent Runner**! Este é um projeto de código aberto sob a licença **GNU AGPLv3**. O objetivo deste projeto não é apenas ser uma ferramenta, mas um **Organismo Coletivo** que evolui com cada tarefa executada.

Existem duas formas principais de contribuir: através de melhorias no **Código** e através do compartilhamento de **Sabedoria**.

---

## 🧠 1. Compartilhando a Sabedoria (Global Memory)

A parte mais valiosa do Agent Runner é o que ele aprende. O arquivo `workspace/memory/global.md` contém a "Memória Viva" do agente. Se você usou o agente e ele aprendeu algo novo, resolveu um erro complexo ou descobriu uma técnica arquitetural superior, nós queremos essa "memória" de volta!

**Como contribuir com memória:**
1.  Localize seu arquivo `workspace/memory/global.md`.
2.  Extraia as lições aprendidas (Seção de "Lições Aprendidas" ou "Diretrizes").
3.  Abra uma Pull Request (PR) adicionando essas lições ao `global.md` do repositório principal.
4.  O Agente usará essa sabedoria coletiva para ser mais inteligente para todos os usuários.

---

## 💻 2. Melhorias no Código (Engine)

Se você encontrou um bug ou tem uma ideia para uma nova funcionalidade no motor do agente:

1.  **Novas Roles**: Adicione novos papéis em `agent/roles/` (ex: SRE, Security Auditor).
2.  **Melhorias nos Scripts**: Otimize os processos em `agent/scripts/`.
3.  **Refinamento de Prompts**: Melhore as instruções em `agent/prompts/` para torná-lo mais preciso.

### Fluxo de Contribuição
1.  Faça um **Fork** do projeto.
2.  Crie uma **Branch** para sua modificação (`git checkout -b feature/minha-melhoria`).
3.  Faça o **Commit** de suas alterações seguindo o padrão (`chore: ...` ou `feat: ...`).
4.  Envie um **Pull Request**.
