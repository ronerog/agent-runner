# Security Policy

## Supported Versions

We actively support the following versions of this project with security updates.

| Version | Supported          |
| ------- | ------------------ |
| main / latest | ✅ |
| older versions | ❌ |

If you are using an unsupported version, please upgrade to the latest version before reporting issues.

---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public issue for security vulnerabilities.**

Instead, please report it through one of the following methods:

- Email: security@yourdomain.com
- GitHub Security Advisory: https://github.com/ronero/agent-runner/security/advisories
- Direct message to the maintainer

Please include as much information as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Possible fix (if known)

---

## Response Process

Once a vulnerability report is received:

1. We will acknowledge receipt within **48 hours**.
2. The vulnerability will be investigated and validated.
3. If confirmed, a fix will be developed and released.
4. The vulnerability may be disclosed publicly after the fix is available.

---

## Responsible Disclosure

We kindly ask researchers and contributors to follow responsible disclosure practices:

- Do not publicly disclose the vulnerability before it is fixed.
- Give maintainers reasonable time to resolve the issue.
- Avoid exploiting the vulnerability beyond what is necessary for demonstration.

---

## Security Best Practices

To help keep the project secure:

- Never commit secrets or API keys
- Use environment variables for sensitive data
- Keep dependencies updated
- Review pull requests carefully

---

Thank you for helping keep this project secure.
